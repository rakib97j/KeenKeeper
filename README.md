
## keenKeeeper | Personal Relationship Management (PRM) Web App
KeenKeeper is a clean, modern Personal Relationship Management (PRM) web application designed to help users systematically track, maintain, and nurture meaningful personal connections and friendships.


## 🛠️ Technologies Use

| Technology | Purpose |
|---|---|
| **Next.js** | Build the UI |
| **App router(Next.js)** | Handle page navigation |
| **Tailwind CSS , Daisy UI** | Styling and responsiveness |
| **Recharts** | Chart |



## 📝 Key Features

## 1.
Friendship Dashboard: Features individual contact profiles showing specific metrics like "Days Since Last Contact," custom communication goals (e.g., "Connect every 14 days"), and dynamic indicators for upcoming interaction due dates.

## 2.
Quick Check-ins: Allows users to log interactions instantly with dedicated action buttons for Calls, Texts, and Video chats.

## 3.
Interaction Timeline: Logs a historical feed of all past communications, categorized by type, to visualize relationship consistency over time.
## 4. 
Friendship Analytics: Utilizes data visualization (such as interactive breakdown charts) to give users clear insights into their interaction habits by contact method.
